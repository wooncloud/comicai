# 화면: 페이지 목록

> v0.1 — 2026-05-16 — Draft

## 1. 목적

프로젝트의 페이지를 추가/순서변경/삭제. 카드 그리드.

## 2. 진입 / 이탈

- 진입: 프로젝트 홈 우측 메인.
- 이탈: 페이지 카드 → 에디터.
- 관련 기능: E-01, E-02.

## 3. 상태

빈 / 정상.

## 4. 와이어프레임

```
┌────────────────────────────────────────────────────────────┐
│ 페이지                                       [+ 새 페이지]  │
├────────────────────────────────────────────────────────────┤
│ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐                         │
│ │ p1 │ │ p2 │ │ p3 │ │ p4 │ │ p5 │  (드래그로 순서)         │
│ │ ⋯  │ │ ⋯  │ │ ⋯  │ │ ⋯  │ │ ⋯  │                         │
│ └────┘ └────┘ └────┘ └────┘ └────┘                         │
└────────────────────────────────────────────────────────────┘
```

## 5. 컴포넌트

드래그 정렬 가능 카드 그리드, `Dialog`(추가/삭제).

## 6. 인터랙션

- "새 페이지" → 크기·비율 프리셋 선택 (웹툰 세로 / A4 / 사용자 정의).
- 카드 ⋯ → 복제 / 삭제.

## 7. 데이터 의존성

- `/v1/projects/:id/pages`.

## 8. 엣지 케이스

- 순서 변경 실패 → 원위치 복귀.
