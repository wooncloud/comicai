# 화면: 페이지 내보내기 (모달)

> v0.1 — 2026-05-16 — Draft

## 1. 목적

페이지를 JPG/PNG로 내보내기. DPI 옵션 제공.

## 2. 진입 / 이탈

- 진입: 에디터 Topbar "내보내기" 버튼.
- 이탈: 다운로드 완료 / 취소.
- 관련 기능: E-11.

## 3. 상태

- 기본 / 변환 중 / 완료 / 실패.

## 4. 와이어프레임

```
┌─────────────────────────────────┐
│ 내보내기                   [✕]  │
├─────────────────────────────────┤
│ 형식:  ( ) PNG   (•) JPG        │
│ DPI:   [   72 ▾]                │
│ 품질:  [████░░░░] 80%           │
│ 범위:  (•) 현재 페이지 ( ) 전체 │
│                                 │
│ [취소]              [내보내기]  │
└─────────────────────────────────┘
```

## 5. 컴포넌트

shadcn `Dialog`, `RadioGroup`, `Slider`, `Select`, `Button`.

## 6. 인터랙션

- "내보내기" → 변환 중 → 다운로드 자동 시작.

## 7. 데이터 의존성

- `POST /v1/pages/:id/export` → 다운로드 URL.

## 8. 엣지 케이스

- 미렌더 패널 있음 → 경고 "n개 패널이 비어있습니다. 계속하시겠습니까?".
- 변환 실패 → 에러 표시.
