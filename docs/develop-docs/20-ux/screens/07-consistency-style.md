# 화면: 일관성 — 그림체

> v0.1 — 2026-05-16 — Draft

## 1. 목적

프로젝트의 그림체 레퍼런스를 등록·관리.

## 2. 진입 / 이탈

- 진입: 프로젝트 홈 사이드바 → "🎨 그림체".
- 이탈: 다른 일관성 탭, 프로젝트 홈, 페이지 에디터.
- 관련 기능: C-01, C-06, C-07.

## 3. 상태

- 빈 / 정상 / 편집 중 / 저장 중.

## 4. 와이어프레임

```
┌────────────────────────────────────────────────────────────┐
│ ← 우주 학교 / 일관성 / 🎨 그림체              [+ 추가]    │
├────────────────────────────────────────────────────────────┤
│ ┌─────────────┐ ┌─────────────┐                            │
│ │ [ref image] │ │ [ref image] │                            │

└────────────────────────────────────────────────────────────┘
```

## 5. 컴포넌트

shadcn `Card`, `Dialog`(편집 모달), `Button`, 이미지 업로더.

## 6. 인터랙션

- "추가": 이름 + 이미지 N장 + 설명 입력.
- "편집": 모달에서 수정 (version +1).
- "삭제": 참조 패널 수 표시 + 확인 모달.

## 7. 데이터 의존성

- `GET/POST/PATCH/DELETE /v1/consistency` (type=style).

## 8. 엣지 케이스

- 사용 중인 항목 삭제 → 경고.
- 이미지 업로드 실패 → 인라인 에러.
