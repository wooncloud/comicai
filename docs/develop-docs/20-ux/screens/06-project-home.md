# 화면: 프로젝트 홈

> v0.1 — 2026-05-16 — Draft

## 1. 목적

프로젝트의 일관성 정보·페이지 목록을 한눈에 보고 작업 진입.

## 2. 진입 / 이탈

- 진입: 대시보드 카드 클릭.
- 이탈: 일관성 정보 화면, 페이지 에디터, 뒤로 대시보드.

## 3. 상태

- 정상 / 로딩.

## 4. 와이어프레임

```
┌────────────────────────────────────────────────────────────┐
│ ← 우주 학교  /  [일관성] [페이지] [⚙]              [👤]   │
├──────────┬─────────────────────────────────────────────────┤
│ 일관성    │ 페이지                          [+ 새 페이지]  │
│          │                                                 │
│ 🎨 그림체 │ ┌────┐ ┌────┐ ┌────┐ ┌────┐                  │
│ 🧑 캐릭터 │ │ p1 │ │ p2 │ │ p3 │ │ p4 │  ...               │
│ 🏞 배경   │ └────┘ └────┘ └────┘ └────┘                  │
│ 🌍 세계관 │                                                 │
│          │ (드래그로 순서 변경)                            │
│ ─────    │                                                 │
│ + 추가   │                                                 │
└──────────┴─────────────────────────────────────────────────┘
```

## 5. 컴포넌트

프로젝트 셸 ([`../components/01-shell-layout.md`](../components/01-shell-layout.md)), 페이지 카드 그리드.

## 6. 인터랙션

- 사이드바 항목 클릭 → 일관성 정보 화면.
- 페이지 카드 클릭 → 에디터.
- 페이지 드래그 → 순서 변경.

## 7. 데이터 의존성

- `GET /v1/projects/:id` + `GET /v1/projects/:id/pages`.

## 8. 엣지 케이스

- 페이지 0개 → 빈 상태 "첫 페이지 추가".
