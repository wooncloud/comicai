# 화면: 일관성 — 세계관

> v0.1 — 2026-05-16 — Draft

## 1. 목적

프로젝트의 세계관 설정(시대, 분위기, 마법 규칙 등)을 텍스트+이미지로 정리.

## 2. 진입 / 이탈

- 진입: 사이드바 → "🌍 세계관".
- 관련 기능: C-04.

## 3. 상태

빈 / 정상 / 편집.

## 4. 와이어프레임

```
┌────────────────────────────────────────────────────────────┐
│ 🌍 세계관                                       [+ 추가]    │
├────────────────────────────────────────────────────────────┤
│ ┌────────────────────────────────────────────────────────┐ │
│ │ 주작 마법학교                                          │ │
│ │ ┌──────┐                                               │ │
│ │ │ 이미지│  근미래, 우주 진출 이후의 마법학교.          │ │
│ │ └──────┘  마법은 양자 기술과 결합되어 있다.            │ │
│ │ v2                                          [편집][삭제]│ │
│ └────────────────────────────────────────────────────────┘ │
└────────────────────────────────────────────────────────────┘
```

## 5. 컴포넌트

shadcn `Card`, `Dialog`, `Textarea`(긴 텍스트), 이미지 업로더.

## 6. 인터랙션

- 캐릭터/배경과 유사.

## 7. 데이터 의존성

- `/v1/consistency` (type=worldview).

## 8. 엣지 케이스

- 텍스트만으로도 등록 가능 (이미지 선택).
