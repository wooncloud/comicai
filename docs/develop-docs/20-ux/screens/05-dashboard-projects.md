# 화면: 대시보드 (프로젝트 목록)

> v0.1 — 2026-05-16 — Draft

## 1. 목적

사용자의 프로젝트 목록을 보여주고 새 프로젝트 생성 / 기존 프로젝트 진입.

## 2. 진입 / 이탈

- 진입: 로그인 후 기본 화면.
- 이탈: 프로젝트 카드 클릭 → 프로젝트 홈. "새 프로젝트" → 생성 모달 → 프로젝트 홈.
- 관련 기능: P-01, P-02.

## 3. 상태

- 빈 상태(첫 사용자) / 정상 / 로딩 / 에러.

## 5. 컴포넌트

shadcn `Card`, `DropdownMenu`(⋯), `Dialog`(생성 모달), `Button`.

## 6. 인터랙션

- 카드 클릭 → 프로젝트 홈.
- ⋯ → 이름 변경 / 삭제(확인 모달).
- "새 프로젝트" → 이름 입력 모달.

## 7. 데이터 의존성

- `GET /v1/projects`.

## 8. 엣지 케이스

- 썸네일 없음 → 플레이스홀더.
- 50개 초과 → 페이지네이션.
