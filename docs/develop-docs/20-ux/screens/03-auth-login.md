# 화면: 로그인

> v0.1 — 2026-05-16 — Draft

## 1. 목적

이메일 또는 OAuth로 로그인.

## 2. 진입 / 이탈

- 진입: 랜딩, 보호 라우트 redirect, 가입 후.
- 이탈: 성공 → return-to URL 또는 `/dashboard`.

## 3. 상태

- 기본 / 입력 중 / 실패 / 잠금(연속 실패).

## 4. 와이어프레임

```
┌──────────────────────────────────┐
│ 로그인                            │
├──────────────────────────────────┤
│ [Google] [GitHub]                │
│ ─────── 또는 ───────             │
│ 이메일    [_____________]        │
│ 비밀번호  [_____________]        │
│                                  │
│ [로그인]                         │
│                                  │
│ 비밀번호 찾기 · 회원가입          │
└──────────────────────────────────┘
```

## 5. 컴포넌트

shadcn `Card`, `Input`, `Button`.

## 6. 인터랙션

- Enter로 제출.
- OAuth 클릭 → `/v1/auth/oauth/:provider`.

## 7. 데이터 의존성

- `POST /v1/auth/login`.

## 8. 엣지 케이스

- 5회 연속 실패 → 5분 잠금 + 이메일 알림.
- OAuth 후 같은 이메일 가입 충돌 → 안내 + 연결 옵션.
