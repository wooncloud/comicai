# 컴포넌트: Toast / Error Card

> v0.1 — 2026-05-16 — Draft

## Toast (가벼운 알림)

```
┌─────────────────────────────────────┐
│ ✅ 저장되었습니다                    │
└─────────────────────────────────────┘
```

- 5초 자동 사라짐.
- 위치: 화면 우하단.

## Error Card (인플레이스, 패널 내부)

```
┌──────────────────────────────────────────────┐
│ ❌ 렌더링 실패                                │
│                                              │
│ API 키가 유효하지 않습니다.                   │
│ Gemini 키를 확인해 주세요.                    │
│                                              │
│ [키 설정으로 이동]  [자세히]  [재시도]        │
└──────────────────────────────────────────────┘
```

- 카테고리별 메시지/액션 다름:
  - `auth` → "키 설정으로 이동"
  - `quota` → "한도 확인"
  - `safety` → "프롬프트 수정"
  - `transient`/`timeout` → "재시도"
  - `invalid` → "입력 확인"
- "자세히" 토글 시 raw 응답 표시 (개발자용).
