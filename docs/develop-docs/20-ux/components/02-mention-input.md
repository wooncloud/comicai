# 컴포넌트: Mention Input

> v0.1 — 2026-05-16 — Draft

## 목적

패널 텍스트 입력란에서 `@`로 일관성 정보를 인용. TipTap 기반.

## 와이어프레임

### 일반 입력

```
┌────────────────────────────────────────────┐
│ 주연이 교실 창가에서 우주를 바라본다       │
└────────────────────────────────────────────┘
```

### 멘션 활성

```
┌────────────────────────────────────────────┐
│ @주│                                        │
└─┬──────────────────────────────────────────┘
  │
  ▼
┌────────────────────────────────┐
│ 🧑 주연        (캐릭터)        │ ← 강조
│ 🌍 주작 마법학교 (세계관)      │
│ 🎨 주조 그림체  (스타일)       │
└────────────────────────────────┘
```

### 멘션 삽입 후 (노드로 표시)

```
┌────────────────────────────────────────────┐
│ [🧑 주연] 가 교실 창가에서 우주를 바라본다 │
└────────────────────────────────────────────┘
```

- 노드는 칩(chip) 형태. 클릭 시 해당 일관성 정보 미리보기 팝오버.
- `Backspace` 한 번으로 노드 전체 삭제.

## 인터랙션

- `@` 입력 → 드롭다운 열림 (현재 프로젝트의 모든 일관성 정보).
- 타입별 아이콘: 🧑(캐릭터) / 🏞(배경) / 🎨(스타일) / 🌍(세계관).
- 화살표 ↑↓, Enter 선택, Esc 취소.
- Fuzzy match: `name` + `aliases`.
- 삭제된 엔티티 노드는 회색 + ⚠ 표시.

## 직렬화

- 저장 형식: TipTap JSON. 노드 attrs에 `id`, `label`, `version`.
- 렌더 시 → [`../../30-tech/05-mention-serialization.md`](../../30-tech/05-mention-serialization.md).
