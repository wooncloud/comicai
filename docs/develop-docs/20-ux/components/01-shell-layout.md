# 컴포넌트: Shell Layout

> v0.1 — 2026-05-16 — Draft

## 목적

앱 전체에서 재사용하는 글로벌/프로젝트 셸 레이아웃. Topbar + (옵션) Sidebar + Content.

## 구성

### 글로벌 셸 (로그인 후 공통)

```
┌──────────────────────────────────────────────────────────┐
│ (A) Topbar (글로벌)                                       │
├──────────────────────────────────────────────────────────┤
│                                                          │
│ (B) Content                                              │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 프로젝트 셸 (페이지 에디터 외 프로젝트 내 화면)

```
┌──────────────────────────────────────────────────────────┐
│ (A) Topbar (프로젝트 컨텍스트)                            │
├──────────┬───────────────────────────────────────────────┤
│ (B)      │                                               │
│ Sidebar  │ (C) Content                                   │
│          │                                               │
└──────────┴───────────────────────────────────────────────┘
```

### 에디터 셸 (페이지 에디터 전용)

```
┌──────────────────────────────────────────────────────────┐
│ (A) Topbar (에디터)                                       │
├──────────┬───────────────────────────┬───────────────────┤
│ (B)      │ (C)                       │ (D)               │
│ Left     │ Center (Canvas)           │ Inspector         │
│ Sidebar  │                           │                   │
└──────────┴───────────────────────────┴───────────────────┘
```

## 컴포넌트 표

| 영역          | 컴포넌트                                      | shadcn                                     |
| ------------- | --------------------------------------------- | ------------------------------------------ |
| (A) Topbar    | 로고, 브레드크럼, 저장상태, 액션, 아바타 메뉴 | `NavigationMenu`, `Avatar`, `DropdownMenu` |
| (B) Sidebar   | 섹션 네비 (일관성/페이지) 또는 리스트         | `Tabs` 또는 자체 List                      |
| (C) Content   | 페이지별 컨텐츠                               | -                                          |
| (D) Inspector | 선택된 객체 속성                              | `Sheet`/`Card` 조합                        |

## 반응형

- ≥1280px: 풀 3-컬럼.
- 1024~1280px: Inspector 너비 축소.
- <1024px: Inspector 토글식. (best-effort)
