# 컴포넌트: Panel Status Badge

> v0.1 — 2026-05-16 — Draft

## 목적

패널의 현재 렌더 상태를 캔버스 위에 항상 표시.

## 상태별 모양

```
┌──────────────┐
│   (대기)     │  ⏳ QUEUED
└──────────────┘

┌──────────────┐
│ ▓▓▓▓░░░░░ 40%│  🔵 RUNNING (progress bar + %)
└──────────────┘

┌──────────────┐
│   (완료)     │  ✅ SUCCEEDED (3초 후 페이드)
└──────────────┘

┌──────────────┐
│   (실패) ↻   │  ❌ FAILED (재시도 버튼)
└──────────────┘

┌──────────────┐
│   (시간초과) │  ⚠ TIMEOUT
└──────────────┘
```

## 위치

패널 도형의 우상단 모서리 inside. 패널이 작을 경우 outside.

## 인터랙션

- 클릭: 잡 상세 모달 (시간, 모델, 에러 메시지, raw 응답 토글).
- 우클릭: 컨텍스트 메뉴 (재시도, 취소, 히스토리 열기).

## 디자인 토큰

색은 [`../02-design-tokens.md`](../02-design-tokens.md) 상태 컬러 사용.
