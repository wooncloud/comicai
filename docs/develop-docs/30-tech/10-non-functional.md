# 비기능 요구사항

> v0.1 — 2026-05-16 — Draft

| 항목              | 목표 (MVP)                | 측정 방법                    |
| ----------------- | ------------------------- | ---------------------------- |
| 동시 렌더링 잡    | 워커 1개당 2~4개          | BullMQ concurrency 설정      |
| 패널 렌더 P95     | < 35s (Gemini 기준)       | `render_duration_seconds`    |
| 자동저장 디바운스 | 1.5s                      | UI 측정                      |
| 첫 화면 FCP       | < 2.5s (홈 네트워크 기준) | Next.js Web Vitals           |
| 가용성            | best-effort (개인 PC)     | uptime monitoring 별도 안 함 |
| 백업 RPO          | 24h                       | 일 1회 dump                  |
| 복구 RTO          | 4h                        | 매뉴얼 절차                  |
| 이미지 용량       | 1프로젝트당 < 2GB 권장    | MinIO usage                  |

## 부하 시나리오 (참고)

- 동시 사용자 10명, 패널 렌더 동시 30건 처리 가능.
- 그 이상은 클라우드 이전 후 확장.
