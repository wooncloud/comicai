## 대표 시나리오: 박지훈, 첫 패널 완성

1. 회원가입 → API 키 등록 (Gemini).
2. 새 프로젝트 "우주 학교".
3. 일관성 정보 등록:
   - 그림체: 가지고 있던 일러스트 1장 업로드.
   - 캐릭터 "주연": 정면 캐릭터 시트 1장, "단발머리, 검은 교복" 설명.
4. 페이지 1 추가 (A4 비율).
5. 페이지 에디터에서 패널 4개 도형 그림.
6. 첫 패널 선택 → 텍스트 입력: `@주연 가 교실 창가에서 우주를 바라본다`.
7. 모델: Gemini 선택 → 렌더 실행.
8. ~25초 후 결과 등장. 마음에 안 들면 텍스트 수정 후 재렌더.

> 성공 기준: 4개 패널에 등장하는 "주연"의 얼굴·헤어스타일이 흔들리지 않는다.
