# 제품 비전

> v0.1 — 2026-05-16 — Draft

## 한 줄 정의

**AI가 만화의 일관성(캐릭터·배경·세계관·그림체)을 유지하며 만화를 그려주는 웹 제작 도구.**

## 핵심 가치 제안

- 작가가 캐릭터/배경/그림체를 한 번 정의하면, AI가 이후 모든 패널에서 그 설정을 유지한 채 그림을 생성한다.
- 패널 단위 정밀 제어(콘티 + 텍스트 + 멘션 + 참조 이미지)로, 단순 텍스트-투-이미지 도구와 차별화한다.

## 타깃 사용자

- 1차: 한국의 개인 창작자(웹툰 지망생, 동인 작가, 1인 만화 제작자).
- 자체 API 키를 발급받아 사용할 수 있는 기술 친화적 얼리어답터.

## 비즈니스 모델

| 단계      | 모델        | 비고                                      |
| --------- | ----------- | ----------------------------------------- |
| MVP       | **BYOK**    | 사용자가 Gemini / OpenAI API 키 직접 등록 |
| Post-MVP  | 크레딧 구매 | 모델별 단가, 1회 렌더당 차감              |
| Long-term | 구독제      | 월정액 + 정량 크레딧                      |

> MVP는 크레딧·결제 인프라까지 데이터 모델로만 준비. 실제 결제는 미연동.

## 비목표 (Out of Scope, MVP)

- 만화 연재/배포 커뮤니티 (Phase 2)
- 다중 사용자 협업 편집
- 모바일 앱 (반응형 웹은 best-effort)
- 자체 GPU 추론 / Stable Diffusion / LoRA 학습
- 해외 시장 / 다국어 UI
