## 핵심 플로우: 가입 → 첫 패널 렌더

```
[랜딩]
   ↓ "시작하기"
[가입 or 로그인] ── OAuth(Google/GitHub)
   ↓
[API 키 등록] (Gemini / OpenAI)
   ↓
[대시보드: 프로젝트 목록]
   ↓ "새 프로젝트"
[프로젝트 홈]
   ↓ 일관성 정보 사이드바
[일관성 정보 편집] ── 그림체/캐릭터/배경/세계관 등록
   ↓ 뒤로
[페이지 추가] (크기·비율 선택)
   ↓
[페이지 에디터]
   ├─ 패널 도형 그리기
   ├─ 패널 선택 → 콘티 / 텍스트(@멘션) / 참조 이미지
   ├─ 모델 선택 → 렌더링
   └─ 결과 확인 → 만족? Y/N
                  ├─ Y: 다음 패널
                  └─ N: 수정 → 재렌더 (히스토리 보존)
   ↓
[페이지 내보내기] (JPG/PNG)
```

## 부가 플로우

### 일관성 정보 수정 → 과거 렌더에 영향?

- 수정 시 `version` +1.
- 이후 렌더링부터 신버전 적용.
- 과거 RenderJob은 IR 스냅샷 보존 → 영향 없음.

### API 키 만료/오류

- 렌더 실패(`Auth` 카테고리) → 토스트 + 키 설정 페이지 링크.
- 5회 연속 Auth 실패 시 키 자동 비활성화.

### 자동 저장

- 편집 변경 후 1.5s 디바운스 → 백그라운드 저장.
- 저장 실패 시 상단 배너 표시.

### 미로그인 접근

- 보호 라우트 진입 시 로그인 페이지로 리다이렉트 (return-to URL 유지).
