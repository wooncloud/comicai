# 화면: 랜딩

> v0.1 — 2026-05-16 — Draft

## 1. 목적
비회원에게 제품 가치를 30초 안에 전달하고 가입/로그인으로 유도.

## 2. 진입 / 이탈
- 진입: 도메인 루트 `/`.
- 이탈: 가입, 로그인, 외부 링크(GitHub, 문의).
- 관련 기능: U-01, U-03.

## 3. 상태
- 비로그인 (기본).
- 로그인 상태 진입 시 → `/dashboard`로 리다이렉트.
