# 화면: 설정 — 프로필

> v0.1 — 2026-05-16 — Draft

## 1. 목적
사용자 표시 이름, 아바타, 이메일 변경.

## 2. 진입 / 이탈
- 진입: 글로벌 Topbar 아바타 메뉴 → "설정".
- 이탈: 저장 후 같은 화면 / 뒤로.

## 3. 상태
보기 / 편집 / 저장 중.

## 4. 와이어프레임

```
┌─────────────────────────────────────────────┐
│ 설정  /  프로필                              │
├─────────────────────────────────────────────┤
│ 아바타:  [👤] [업로드] [기본으로]           │
│ 이름:    [김주연______]                     │
│ 이메일:  dntjdgh02@gmail.com (변경 [✏])     │
│                                             │
│ [저장]                                       │
└─────────────────────────────────────────────┘
```

## 5. 컴포넌트
shadcn `Card`, `Input`, `Avatar`, `Button`.

## 6. 인터랙션
- 이메일 변경 → 새 이메일로 인증 메일 발송.

## 7. 데이터 의존성
- `PATCH /v1/me`.
