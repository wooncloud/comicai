# 화면: 회원가입

> v0.1 — 2026-05-16 — Draft

## 1. 목적
이메일·비밀번호 또는 OAuth로 신규 계정 생성.

## 2. 진입 / 이탈
- 진입: 랜딩 CTA, 로그인 화면 하단 링크.
- 이탈: 가입 성공 → 이메일 인증 안내 → 로그인.
- 관련 기능: U-01, U-03.

## 3. 상태
- 기본 / 입력 중 / 검증 실패 / 제출 중 / 성공.

## 6. 인터랙션
- 클라이언트 검증: 이메일 형식, 비밀번호 10자+영문/숫자, 일치 여부.
- 제출 시 로딩 → 성공 메시지 → 이메일 인증 안내 표시.

## 7. 데이터 의존성
- `POST /v1/auth/signup`.

## 8. 엣지 케이스

- 약관 미동의 → 버튼 비활성화.
