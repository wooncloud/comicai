# 화면: 일관성 — 캐릭터

> v0.1 — 2026-05-16 — Draft

## 1. 목적
프로젝트의 등장 인물(캐릭터)을 등록·관리. 이름·별칭·설명·캐릭터 시트 N장.

## 2. 진입 / 이탈
- 진입: 사이드바 → "🧑 캐릭터".
- 이탈: 다른 탭.
- 관련 기능: C-02, C-06, C-07.

## 3. 상태
- 빈 / 정상 / 편집 / 저장 중.

## 4. 와이어프레임

```
┌────────────────────────────────────────────────────────────┐
│ 🧑 캐릭터                                       [+ 추가]    │
├────────────────────────────────────────────────────────────┤
│ ┌────────────────────────────────┐                          │
│ │ [정면][측면][표정 1]...        │                          │
│ │ 주연                            │                          │
│ │ 별칭: 김주연, 주                │                          │
│ │ 단발머리, 검은 교복, 빨간 머리핀│                          │
│ │ v3   [편집] [삭제]              │                          │
│ └────────────────────────────────┘                          │
└────────────────────────────────────────────────────────────┘
```

## 5. 컴포넌트
shadcn `Card`, `Dialog`(편집), `Input`, `Textarea`, 멀티 이미지 업로더.

## 6. 인터랙션
- 추가/편집 모달에서 이미지 N장, 이름·별칭(쉼표 구분)·설명.
- 별칭은 @멘션 자동완성 검색 대상.

## 7. 데이터 의존성
- `GET/POST/PATCH/DELETE /v1/consistency` (type=character).

## 8. 엣지 케이스
- 캐릭터 시트 0장도 허용(텍스트만으로 시도 가능). 권장 1장 이상.
- 삭제 시 참조 패널 수 경고.
