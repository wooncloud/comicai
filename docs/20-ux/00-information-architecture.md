# Information Architecture (IA)

> v0.1 — 2026-05-16 — Draft

## 사이트맵

```
[랜딩]
  ├── [회원가입]
  ├── [로그인]
  │    └── [OAuth 콜백]
  │
  └── [대시보드] (로그인 후)
       ├── [프로젝트 목록]
       │    └── [프로젝트 홈] (선택 시)
       │         ├── [일관성 정보]
       │         │    ├── 그림체
       │         │    ├── 캐릭터
       │         │    ├── 배경
       │         │    └── 세계관
       │         └── [페이지 목록]
       │              └── [페이지 에디터] ⭐ 핵심
       │                   └── [페이지 내보내기] (모달)
       │
       └── [사용자 설정]
            ├── 프로필
            ├── API 키 관리 (BYOK)
            └── 계정/보안
```
