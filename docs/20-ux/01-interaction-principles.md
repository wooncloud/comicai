# 인터랙션 원칙

> v0.1 — 2026-05-16 — Draft

## 1. 상태 가시성
- 모든 비동기 작업은 상태 표시 (대기/진행/완료/실패).
- 패널 상단 우측에 상태 뱃지 항상 노출.
- 자동 저장 결과는 Topbar 우측에 "저장됨 · n초 전".

## 2. 비차단 (Non-blocking)
- 렌더링 실행 후에도 다른 패널 편집 가능.
- 한 패널의 실패가 다른 패널에 영향 없음.
- 무거운 작업은 토스트가 아니라 인플레이스 상태로 표시.

## 3. 직접 조작
- 패널은 드래그로 그리고, 드래그로 이동·리사이즈.
- 멘션은 `@` 입력 → 드롭다운 → Enter/클릭.
- 이미지는 드롭으로 업로드.

## 4. 되돌리기 (Undo/Redo)
- 캔버스 편집 단위: `Cmd/Ctrl+Z`, `Cmd/Ctrl+Shift+Z`.
- 텍스트 입력 단위: TipTap 기본 undo.
- 렌더 결과 되돌리기는 별도 "히스토리 트레이"에서 명시적으로.

## 5. 자동 저장
- 변경 후 1.5s 디바운스 → 백그라운드 저장.
- 저장 실패 → Topbar 빨간 배너 + 재시도 버튼.
- 같은 사용자가 다른 탭에서 열면 read-only 잠금 (Q-07).

## 6. 키보드 단축키 (페이지 에디터)

| 키 | 동작 |
|---|---|
| `V` | 선택 도구 |
| `P` | 패널 그리기 도구 |
| `D` | 콘티 스케치 도구 |
| `T` | 텍스트 입력 도구 |
| `Enter` (패널 선택 중) | 렌더링 실행 |
| `Cmd/Ctrl+S` | 즉시 저장 |
| `Cmd/Ctrl+Z` | Undo |
| `Cmd/Ctrl+Shift+Z` | Redo |
| `Esc` | 도구 → 선택, 모달 닫기 |

## 7. 접근성
- 모든 인터랙션 요소는 키보드만으로도 도달 가능.
- 대비비 WCAG AA 이상.
- 색상만으로 상태 구분 금지 → 아이콘+텍스트 동반.

## 8. 빈 상태(Empty State)
- 모든 목록 화면은 "처음 만들기" CTA 포함.
- 일러스트보다 명확한 액션 우선.

## 9. 에러 메시지
- 사용자 행동을 유도하는 문구.
  - ✗ "Internal Server Error"
  - ✓ "잠시 후 다시 시도해 주세요. 계속되면 API 키를 확인하세요."
