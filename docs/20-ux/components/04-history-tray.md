# 컴포넌트: History Tray

> v0.1 — 2026-05-16 — Draft

## 목적
선택된 패널의 렌더 히스토리(최대 20)를 미리보기 + 복원.

## 와이어프레임

```
┌────────────────────────────────────────────┐
│ History (패널 #3)                  [✕]    │
├────────────────────────────────────────────┤
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐       │
│  │ ◉│ │  │ │  │ │  │ │  │ │  │ │  │       │
│  └──┘ └──┘ └──┘ └──┘ └──┘ └──┘ └──┘       │
│   현재  -1   -2   -3   -4   -5   ...        │
│                                            │
│  선택 항목 정보:                            │
│  · 모델: Gemini                             │
│  · 생성: 2026-05-16 14:23                   │
│  · 상태: SUCCEEDED                          │
│  · 프롬프트: "주연이 교실에서..."           │
│                                            │
│  [복원하기]  [삭제]                         │
└────────────────────────────────────────────┘
```

## 인터랙션
- 썸네일 클릭: 선택 + 정보 패널 갱신.
- "복원": 해당 잡의 결과를 현재로 설정. 기존 현재는 큐에 남음.
- "삭제": 해당 잡 영구 삭제 (확인 모달).
- 큐 가득(20) 상태에서 새 렌더 → 가장 오래된 항목 자동 제거 표시.

## 빈 상태
"아직 렌더링 기록이 없습니다. 우측 Inspector에서 렌더를 실행하세요."
